# Template

Place your QSL card PDF here as `qsl-card-template.pdf`
(or whatever path you set in `config.php` under `template_pdf`).

The bundled `qsl-card-template.pdf` is a sample landscape A4 design.
The field coordinates in `generate.php` (`generateQslPdf()`) are calibrated for it.
If you swap in a different template, adjust the `SetXY()` values accordingly.
